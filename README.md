# Sorting-Algorithms
For C# .Net

Name Space For Dll:   "using SortingAlgorithm;"

Class Name: "Sorting"

Methods: All methods are static on class Sorting. Call it by Sorting.algorithm_name.

Below Algorithms are implemented with overload for type int, float and double.
1, Bubble Sort
2, Insertion Sort
3, Quick Sort
4, Counting Sort
5, Heap Sort
6, Merge Sort
7, Selection Sort
8, Shell Sort
